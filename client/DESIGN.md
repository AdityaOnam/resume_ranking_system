# Resume Analyzer Design System

This document provides design tokens for the AI UI generator (Google Stitch).

## Colors
- **Primary:** `#3B82F6` (Blue 500)
- **Secondary:** `#10B981` (Emerald 500)
- **Background (Dark):** `#0F172A` (Slate 900)
- **Surface (Dark):** `#1E293B` (Slate 800)
- **Text (Primary):** `#F8FAFC` (Slate 50)
- **Text (Secondary):** `#94A3B8` (Slate 400)
- **Success:** `#22C55E` (Green 500)
- **Warning:** `#F59E0B` (Amber 500)
- **Error:** `#EF4444` (Red 500)

## Typography
- **Font Family:** 'Inter', sans-serif
- **Headings:** Bold, clean, minimal tracking
- **Body:** Regular weight, high legibility

## Spacing & Layout
- Based on 4px grid (e.g., 4, 8, 16, 24, 32)
- Use standard Tailwind classes (`p-4`, `m-4`, `gap-4`)
- Max-width containers for readability

## Vibe & Feel
- **Aesthetic:** Modern, Dark Mode, Premium, Glassmorphism elements
- **Interactions:** Subtle hover effects (e.g., `hover:bg-slate-700`, `transition-all`)
- **Shapes:** Slightly rounded corners (`rounded-lg` or `rounded-xl`)
